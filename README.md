# Skill Forge - 将知识锻造成装备

把学习变成 RPG 游戏的单页应用。上传 PDF/Markdown 教材，AI 自动生成技能树，刷题闯关打 Boss，爆装备升级。

## 功能

- 🗺️ **技能树** — AI 分析教材生成网状知识图谱
- 📚 **分段学习** — 每个知识点拆成递进课时 + 测验
- ⚔️ **装备系统** — 6 种稀有度，4 个装备栏，随机属性
- 💀 **Boss 战** — 宝可梦式回合制战斗
- 📋 **任务系统** — 动态生成的主线任务
- 📅 **每日签到** — 日历打卡 + 补签
- 🔄 **设备同步** — 电脑和手机通过局域网实时同步进度

## 使用

直接浏览器打开 `skill-forge.html`，配置任意 OpenAI 兼容 API 即可使用。

```bash
# 双设备同步（电脑运行中继，手机浏览器连电脑 IP）
npm install && node relay-server.js
```

## 技术

- React 18 + Tailwind CSS + Dexie.js
- 纯前端，无构建工具，单 HTML 文件
- WebSocket 局域网同步
